# Hitster Clone - Roadmap & TODO

## 1. UI & UX Improvements
- [ ] **Loading & API Activity:** Display clear loading states or spinners during API fetches so players aren't left waiting without visual feedback.
- [ ] **Enhanced Feedback:** Provide clearer, more satisfying visual and audio feedback for correct/incorrect guesses.
- [ ] **Instructions:** Add a brief tutorial or instruction overlay for new players.

## 2. Game Configuration Options
- [ ] **Custom Playlists/Filters:** Allow players to choose specific song types, genres, or limit the game to specific decades (e.g., 80s only, Rock only).

## 3. Local Multiplayer
- [ ] **Pass-and-Play Mode:** Support multiple players taking turns on the same device, keeping track of individual scores.

## 4. Physical Game Support
- [ ] **Printable Cards:** Create a feature to generate and print physical song cards.
- [ ] **QR Code Integration:** Embed QR codes on the back of printed cards that link directly to the song's playback for the "card reader" to play without seeing the details.

## 5. Multi-Device Online Multiplayer
- [ ] **Backend Server:** Build a real-time backend (e.g., using WebSockets/Socket.io or Firebase) to sync game state across different devices.
- [ ] **QR Code Join:** Allow a host to display a QR code on their screen that friends can scan with their phones to instantly join the game session.
